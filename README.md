# 🏦 Python ATM Simulator

A simple ATM Simulator built with Python. This project demonstrates the use of variables, loops, conditional statements, exception handling, and user input validation.

## Features

- User information input
- Withdrawal system
- Balance verification
- Invalid amount handling
- Maximum 3 attempts
- Exception handling
  - ValueError
  - KeyboardInterrupt
- Transaction receipt generation

## Technologies Used

- Python 3

## Concepts Practiced

- Variables
- Input/Output
- Conditional Statements
- Loops
- Exception Handling
- Boolean Variables
- Basic Banking Logic
